[[ats_core_integrity]]
====
[%metadata]
label:: /conf/core/integrity
subject:: /req/core/integrity
test-purpose:: Validate that a WNM has a valid integrity object.

[.component,class=test method]
=====
[.component,class=step]
--
Check for the existence of an `+integrity+` object in the WNM.
--

[.component,class=step]
--
If an `+integrity+` object is found:

* check that the `+method+` property exists and whose value is one of the following:
** sha256
** sha384
** sha512
** sha3-256
** sha3-384
** sha3-512
* check that the `+value+` property is base64 encoded:
** base64 decode the `+value+` property
** base64 encode the resulting value of the previous step
** verify that the original `+value+` property and decoded/encoded value are identical
